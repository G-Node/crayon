documentation examples contribute

crayon.js is a small, free, JavaScript library based on jQuery, d3 and SVG for easily creating dynamic plots of time-series (charts).

Dynamic. We mean it.

You can add more data to the charts at runtime. If the domain or range of the functions needs to be adjusted, they will be! The grids and the signals etc. will be readjusted to the new axis values.

Dependencies and browser support

Depends on jQuery and d3. This section is under development.
